# Movie Ticket Console

## Project Description

`Movie Ticket Console` is a console-based cinema ticket management project written in C. It simulates the main workflows of a movie ticket system in a terminal application, including authentication, movie management, screening management, ticket booking, ticket cancellation, purchase history tracking, and manager reporting.

The application is organized into separate model, repository, service, and utility layers, with CSV files used for persistent data storage. This keeps the codebase modular, easier to maintain, and straightforward to build with CMake.

## Project Structure

```text
movie-ticket-console/
|-- data/                    # CSV data files
|-- include/
|   |-- models/              # Struct definitions and model headers
|   |-- repos/               # Repository layer headers
|   |-- services/            # Service layer headers
|   |-- utils/               # Utility headers
|   `-- constanst.h          # Shared constants
|-- src/
|   |-- repos/               # CSV-backed data access logic
|   |-- services/            # Business logic and user workflows
|   |-- utils/               # Input, menu, CSV, and helper utilities
|   `-- main.c               # Program entry point
|-- build.ps1                # Windows build, export, and run script
|-- CMakeLists.txt           # CMake build configuration
`-- README.md
```

## Build and Run on Windows

1. Clone the repository:

```powershell
git clone https://github.com/CQ-66-CNTT-Unknown/movie-ticket-console.git
cd movie-ticket-console
```

2. Run the PowerShell build script:

- Build the project into `build/`
```powershell
./build.ps1
```

- Remove existing `build/` and `dist/`, then rebuild
```powershell
./build.ps1 -Clean
```

- Build and export the executable with CSV data into `dist/`
```powershell
./build.ps1 -Export
```

- Build, export, and run the application from `dist/`
```powershell
./build.ps1 -Run
```

3. Output locations:

- `build/movie-ticket-console.exe`: executable generated by CMake
- `dist/movie-ticket-console.exe`: exported executable created by `-Export` or `-Run`
- `dist/data/`: CSV data files copied for runtime use

4. Prerequisites:

- `cmake`
- `ninja`
- `gcc`
- `PowerShell`
